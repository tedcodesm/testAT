# Africa's Talking SMS Demo

A simple Node.js application demonstrating how to send SMS messages using the Africa's Talking SMS API. This project is designed for beginners who want to learn the basics of integrating SMS functionality into their applications.

## Features

* Send SMS messages using the Africa's Talking API
* Environment variable configuration for secure credential management
* Simple and easy-to-understand code structure
* Basic error handling and API response logging

## Tech Stack

* Node.js
* Express.js
* Africa's Talking SDK
* dotenv

## Purpose

This project serves as a minimal example for developers learning how to work with Africa's Talking SMS services. It focuses on the core SMS functionality without the complexity of a full production system.
